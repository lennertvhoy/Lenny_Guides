# Termux

Termux is een Linux-achtige terminalomgeving voor Android.

## Wat Termux doet

Geeft je een SSH-client en Git-tools op je telefoon.

## Wanneer te gebruiken

- Snelle statuscontroles
- Korte goedkeuringen
- Handoffs onderweg lezen

## Basisinstallatie

Installeer via F-Droid en voer daarna uit:

```bash
pkg install -y openssh git nano
```

## Veiligheidsopmerkingen

!!! warning "Telefoon is voor kleine taken"
    Grote diffs en veiligheidsbeslissingen horen op een laptop.

## Veelgemaakte fouten

- Een verouderde Termux-versie gebruiken uit een verouderde appstore
- Complexe commando's typen op een virtueel toetsenbord
