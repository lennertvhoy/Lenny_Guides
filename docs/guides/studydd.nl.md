# StudyDD

StudyDD is een repo-eerst leerworkflow.

## Wat StudyDD doet

Slaat op wat je leert, je zwakke punten, bronnen, reviews en bewijs in de repo op.

## Wanneer te gebruiken

- Een nieuwe taal leren
- Een nieuw framework leren
- Een nieuw concept leren met een AI-tutor

## Basisinstallatie

Voeg deze bestanden toe aan je leerrepo:

```bash
AGENTS.md
STATUS.md
NEXT_ACTIONS.md
BACKLOG.md
WORKLOG.md
PROJECT_STATE.yaml
SOURCES.md
REVIEW.md
evidence/README.md
```

## Veiligheidsopmerkingen

!!! warning "Beheersing vereist bewijs"
    Markeer een onderwerp niet als beheerst zonder een test, quiz of geproduceerd artefact.

## Veelgemaakte fouten

- De agent REVIEW.md laten bijwerken zonder je eigen review
- SOURCES.md overslaan
