# Termux

Termux is a Linux-like terminal environment for Android.

## What Termux does

Gives you an SSH client and Git tooling on your phone.

## When to use it

- Quick status checks
- Short approvals
- Reading handoffs on the go

## Basic setup

Install from F-Droid, then run:

```bash
pkg install -y openssh git nano
```

## Safety notes

!!! warning "Phone is for short tasks"
    Large diffs and security decisions belong on a laptop.

## Common mistakes

- Using an outdated Termux version from a stale app store
- Typing complex commands on a virtual keyboard
