# StudyDD

StudyDD is a repo-first learning workflow.

## What StudyDD does

Stores what you learn, your weak points, sources, reviews, and evidence in the repo.

## When to use it

- Learning a new language
- Learning a new framework
- Learning a new concept with an AI tutor

## Basic setup

Add these files to your study repo:

```bash
AGENTS.md
STATUS.md
NEXT_ACTIONS.md
BACKLOG.md
WORKLOG.md
PROJECT_STATE.yaml
SOURCES.md
REVIEW.md
evidence/README.md
```

## Safety notes

!!! warning "Mastery needs evidence"
    Do not mark a topic mastered without a test, quiz, or produced artifact.

## Common mistakes

- Letting the agent update REVIEW.md without your review
- Skipping SOURCES.md
